# Finch Collector

Finch Collector is an app that allows users to add finches, each added finch has a feeding time and seeds that they enjoy to eat.

![](https://i.imgur.com/wEVxcSh.png)

## Technologies Used

- Python
- Django
- PostgreSQL
- AWS
- Materialize
- JavaScript
- HTML
- CSS
